# KK Quiz

KK Quiz — универсальный модуль квизов для 1C-Битрикс.

## Технические требования

- PHP 8.2+
- JavaScript ES6+
- 1C-Битрикс с поддержкой D7
- Модуль iblock

## Архитектура

- 1 инфоблок для структуры квизов
- 1 инфоблок для заявок
- раздел инфоблока = отдельный квиз
- элемент QUESTION = вопрос
- элемент RESULT = результат
- ответы вопроса будут храниться через кастомное свойство KK_ANSWERS
