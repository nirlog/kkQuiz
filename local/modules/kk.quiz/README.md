# KK Quiz

KK Quiz — универсальный модуль квизов для 1C-Битрикс.

## Технические требования

- PHP 8.2+
- JavaScript ES6+
- 1C-Битрикс с поддержкой D7
- Модуль iblock

## Архитектура

- 1 инфоблок для структуры квизов
- 1 инфоблок для заявок
- раздел инфоблока = отдельный квиз
- элемент QUESTION = вопрос
- элемент RESULT = результат
- ответы вопроса будут храниться через кастомное свойство KK_ANSWERS

## Стандарт разработки

- PHP 8.2+
- JavaScript ES6+
- 1C-Битрикс D7
- Основная бизнес-логика должна быть в D7-стиле:
  - namespace Kk\Quiz
  - autoload через Loader::registerAutoLoadClasses
  - сервисы в /lib/Service
  - репозитории в /lib/Repository
  - AJAX-контроллеры через Bitrix\Main\Engine\Controller
- Legacy API допускается только изолированно:
  - установка инфоблоков
  - создание свойств
  - создание пользовательских полей
  - операции, где D7 API неполный или неудобный
- Компоненты и фронтенд не должны напрямую работать с CIBlockElement/CIBlockSection.
  Доступ к данным должен идти через Repository-классы.

## Свойство KK_ANSWERS

KK_ANSWERS — кастомное свойство инфоблока для хранения вариантов ответа вопроса.
Редактируется в админке как карточки ответов, внутри хранится технический JSON.
Связанные вопросы и результаты выбираются из текущего раздела-квиза, а для картинки ответа показывается превью по ID файла.
Для варианта ответа можно загрузить изображение. В JSON хранится ID файла Битрикса в поле image_id.
